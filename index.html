SummAI Case Study

The Challenge

Information overload is a major challenge for students and professionals. The goal was to create a unified platform that could quickly ingest and process various media formats (text, audio, video) and provide concise summaries and accurate answers to user queries.

My Solution

I built an AI-driven platform leveraging a microservices-style architecture. It uses LangChain to create robust data processing pipelines, integrating Whisper for audio transcription and various Groq models (like LLaMA 3 and Mixtral) for summarization and Q&A. The solution achieves a 3-second average processing time and supports a context of over 128k tokens.

Tech Stack

Python LangChain Streamlit Groq API Whisper

Amazon ML Challenge Case Study

The Challenge

The task was to accurately extract structured data (product quantities and units) from unstructured text found in images of receipts and invoices. This required overcoming challenges like varied image quality, different fonts, and inconsistent text formatting.

My Solution

I designed a hybrid system combining multiple OCR engines (Tesseract and EasyOCR) to maximize text extraction accuracy. The extracted text was then fed into a fine-tuned LLaMA 3.1 8B model. The model was specifically trained to identify and predict quantities and units, resulting in a robust solution that placed in the top 200 among over 75,000 participants.

Tech Stack

Python PyTorch LLaMA 3.1 Tesseract EasyOCR Fine-tuning
